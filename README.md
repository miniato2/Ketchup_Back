## 프로젝트 소개

Ketchup은 조직 내에서 효율적인 커뮤니케이션과 협업을 지원하는 그룹웨어 프로그램 입니다.  
조직 구성원들이 하나의 플랫폼에서 정보 공유, 일정 관리, 파일 공유, 메신저, 전자 결재 등을 수행할 수 있습니다. 이 프로그램은 업무 효율을 극대화하고, 의사소통의 투명성을 확보하는 데 목적이 있습니다.
